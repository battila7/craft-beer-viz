Help!

If you're feeling lost, then fear not! This little guide will help you discover the functionality of Craft Beer Viz.

First things first: this visualization is not about craft beer consumption, but craft beer production.

Craft Beer Viz offers a country and a state view. In addition, the country view has four different visualization modes. Each of these views and modes are described by the cards below.

Scroll down!
expand_more
The Beginnings...

This is the screen you're presented with when opening up Craft Beer Viz. It's called the country view, but currently it's pretty blank. There are two things you can do: clicking on a state will open up the state view, while in the menu (button in the bottom right corner) you can choose a visualization mode for the country view.

Colored Visualization Mode

There are two visualization modes that operate with map coloring: Number of Breweries and Number of Beers. The darker a state is, the more breweries/beers it has. You can reveal the actual number by hovering over a state with the mouse.

Labeled Visualization Mode

We also have two visualization modes utilizing labels: Most Popular Type and Moust Popular Nationality. The former places a label on a state displaying which type of beer is brewed the most, while the latter uses the labels to describe the most brewed nationality.

State View – Statistics

Clicking on a state in country view will take you into the state view. On the left the state is shown in large with red circles marking cities with breweries. The panel on the right displays the share of types and nationalities in the total number of beers brewed in the selected state.

State View – Breweries

If you hover over a red circle, a label is shown with the name of the city under the mouse. Clicking on the circle will reveal the breweries of the selected city on the right panel.
